# Demacia Shop Client (React + Vite)

## Setup

1. Copy `.env.example` to `.env` and set your API key:

```
cp .env.example .env
```

Edit `.env` and set:

```
VITE_GEMINI_API_KEY=your_api_key_here
```

2. Install dependencies and start the dev server:

```
npm install
npm run dev
```

## Build

```
npm run build
```
